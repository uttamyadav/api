These third-party services may collect certain data to function effectively. Below is a breakdown of the data collected, its purpose, and how it is used.

Data Collected:

Purpose of Collection:

  • Monitor app performance and user engagement

The collected data is aggregated for analytical purposes and does not personally identify users.

2. AdMob (Google Ad Services)

Data Usage:

Personalized ads are based on user behavior and preferences.

3. Firebase Crashlytics

Data Collected:

  • Crash logs and diagnostic data

Purpose of Collection:

  • Identify and diagnose app crashes
  • Improve app stability and fix bugs

Data Usage:

The data collected is used strictly for debugging and crash analysis.

Permissions Required

Internet Permission

Purpose: Used to connect the app to the internet for remote control and streaming features.
Details: Allows connection with your PS5 console and fetches required data for remote access. No personal browsing data is collected.

Purpose: Checks your device's network status to ensure a stable connection.
Details: Determines if connected via Wi-Fi or mobile data for optimized functionality.
